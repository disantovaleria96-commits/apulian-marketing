# Apulian Marketing — Sito web

Sito statico in HTML / CSS / JavaScript vanilla. Nessun build step.

## Struttura

```
.
├── index.html          # Pagina principale
├── css/styles.css      # Stili (design system + responsive)
├── js/main.js          # Interazioni (menu, scroll, animazioni, form)
├── vercel.json         # Configurazione Vercel (cache headers)
└── .gitignore
```

## Sviluppo locale

Aprire `index.html` in un browser, oppure servirlo con un qualsiasi static server.

## Deploy

Il sito è pubblicato su Vercel. Ogni push sul branch `main` triggera un deploy automatico.
